# Product Requirements Document: Mahjong Docker

## 1. Vision
To create the most accessible and technically robust open-source Mahjong platform that serves as both a playable game and a reference implementation for complex board game logic.

## 2. Target Audience
- Mahjong enthusiasts seeking a clean, standard-compliant online experience.
- Developers looking for a reference implementation of state-machine-driven multiplayer games.

## 3. Key Features
- **Real-time Table View**: A responsive SVG-based game board showing discards, melds, and player status.
- **Interactive Hand Management**: Click-to-play interface with clear separation of drawn tiles.
- **Dynamic Action System**: Contextual buttons for "Win," "Pong," "Kong," and "Chi" that appear only when valid.
- **Room Lobby**: A pre-game area for player coordination, ready-status toggling, and nickname customization.
- **Visual Themes**: Support for Light and Dark modes with high-contrast tile graphics.
